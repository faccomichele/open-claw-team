## Research Sources

Preferred primary sources (bookmark these patterns):
- Competitor pricing: `<competitor>.com/pricing`
- Market reports: Statista, IBISWorld, McKinsey Global Institute, CB Insights
- Funding data: Crunchbase, PitchBook
- Job postings: LinkedIn, Greenhouse, Lever (signals company direction)
- Technical docs: GitHub repos, developer portals, changelogs
- Regulatory: gov.uk, FTC, EU Publications Office

## Confidence Scale

Use this consistently across all reports:
- **High confidence** — Multiple independent sources confirm; direct data available
- **Moderate confidence** — 2+ sources, some extrapolation required
- **Low confidence** — Single source or significant extrapolation; flag explicitly
